# TESBDII Backend Test

## Description
Project ini dibuat sebagai bagian dari tes backend **TESBDII**.  
Backend menggunakan **Node.js**, **Express**, dan **PostgreSQL**.

---

## Prerequisites
- Node.js >= 18.x  
- NPM >= 9.x  
- PostgreSQL >= 15.x  

---

## Installation

1. Clone repository
```bash
git clone https://github.com/Ridwan731911/TestBEDII.git
cd TestBEDII
npm install
cp .env.example .env
````
